## Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
3. Framework Preset: **Other**
4. Root Directory: leave as `/` (or `blackvale/` if you pushed just this folder)
5. Click **Deploy** — done.
